# ChatFlow Terminal Client

A simple AI-powered chat application built with .NET Console App that connects to multiple AI models through the AgentRouter API gateway.

---

## ✨ Features

- **Real-time interaction:** Chat with AI models directly from your terminal.
- **Multi-model support:** Compatible with various AI models (Claude, DeepSeek, GPT, etc.).
- **Clean Code:** Utilizes `HttpClient` for efficient and organized HTTP communication.
- **Data handling:** Uses `System.Text.Json` for high-performance JSON serialization.

---

- .NET SDK 6.0 or later.
